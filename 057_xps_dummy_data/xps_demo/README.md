# PET フィルム C1s XPS ダミーデータセット

社内講習会「生成AIを使ったMI」**デモ3**向けの合成（架空）XPS データセットです。
「生の測定データから MI 用特徴量を生成する」デモの入力として設計しています。

> ⚠️ 数値は材料科学的に「それらしい」オーダーで作った**完全な架空データ**であり、
> 実測値ではありません。教材デモ専用です。

---

## 1. 概要

表面処理条件の異なる PET フィルムについて、**50 個のダミー C1s XPS スペクトル**を
生成します。後続の Claude Code デモで以下を行えるように設計しています。

- CSV スペクトルの一括読み込み
- バックグラウンド補正
- 3 成分へのピーク分離（C–C/C–H, C–O, O–C=O）
- ピーク位置・面積・面積比・FWHM の抽出
- フィッティング不良データの検出
- 表面処理条件との関係の可視化
- **真値との比較による解析精度の検証**

生成に使った真のパラメータは、解析対象とは分離した `ground_truth/` に保存しています。
フィッティングコードからは参照せず、答え合わせ用として使ってください。

## 3. データ仕様

### 測定領域

| 項目 | 設定 |
| --- | --- |
| 結合エネルギー範囲 | 292.0 〜 282.0 eV |
| データ間隔 | 0.05 eV |
| データ点数 | 201 点 |
| スペクトル数 | 50 |
| ピーク数 | 原則 3 成分 |
| 横軸方向 | 高結合エネルギー → 低結合エネルギー（降順で保存） |

CSV は XPS 表示に合わせ結合エネルギーを**降順**で保存しています。解析側では
必要に応じて昇順に並べ替えてください。

### ピーク成分

| 成分 | 基準ピーク位置 | 帰属 |
| --- | ---: | --- |
| Peak 1 | 284.8 eV | C–C / C–H |
| Peak 2 | 286.3 eV | C–O |
| Peak 3 | 288.7 eV | O–C=O |

ピーク位置は試料ごとに小さく変動させています。

### 試料構成（5 群 × 各 10 反復）

| グループ | 処理条件 | 出力 | 処理時間 | 試料 |
| --- | --- | ---: | ---: | ---: |
| G1 | 未処理 | 0 W | 0 s | PET_001–010 |
| G2 | 弱い O₂ プラズマ | 50 W | 10 s | PET_011–020 |
| G3 | 中程度 O₂ プラズマ | 100 W | 30 s | PET_021–030 |
| G4 | 強い O₂ プラズマ | 150 W | 60 s | PET_031–040 |
| G5 | 非常に強い O₂ プラズマ | 200 W | 120 s | PET_041–050 |

処理が強くなるほど **C–C/C–H の面積比が減少**し、**C–O と O–C=O が増加**します
（`preview/true_area_fraction_by_group.png` 参照）。各試料には中心値まわりの
ランダムばらつきを加えています。

### スペクトル合成モデル

```text
観測強度 = 3 個の面積規格化 Pseudo-Voigt ピーク
        + Shirley 型に近いバックグラウンド（定数 + 一次 + 二次 + 累積）
        + ベースラインドリフト
        + カウントノイズ + Gaussian ノイズ
        + 必要に応じた異常成分
```

- ピーク形状は**面積規格化 Pseudo-Voigt**。面積を直接パラメータに与えるため、
  真の面積とフィッティング面積を直接比較できます。
- スペクトル全体に共通する**帯電エネルギーシフト**を加えています
  （主ピークを 284.8 eV に合わせる帯電補正のデモに使えます）。

## 4. 意図的な異常データ（5 試料）

`ground_truth/anomaly_labels.csv` に正解ラベルを保存しています
（`metadata.csv` には記載していません）。

| sample_id | 異常タイプ | 内容 |
| --- | --- | --- |
| PET_007 | high_noise | 通常の約 3 倍のノイズ |
| PET_018 | spike | 数点に大きな外れ値 |
| PET_026 | low_signal | 全体強度が通常の約 30% |
| PET_033 | charging_shift | 約 0.3 eV の帯電シフト |
| PET_044 | unmodeled_shoulder | 285.5 eV 付近に弱い第 4 成分（3 ピークモデル外） |

これにより、残差が大きい／制約端に張り付く／不自然に広い／スパイク除去が必要／
3 ピークで説明できない、といったケースを解析デモで扱えます。

## 6. 再生成・検証の方法

```bash
# 依存パッケージ: numpy, pandas, scipy, matplotlib, pyyaml
cd xps_demo

# 1) データ生成（乱数シードにより完全再現）
python scripts/generate_xps_data.py

# 2) 検証 + プレビュー画像生成
python scripts/validate_generated_data.py
```

`generate_xps_data.py` は既存の元データを上書きして再生成します
（同じシードなので内容は同一）。生成条件を変えたい場合は
`config/generation_config.yaml` を編集してください。乱数シードは同ファイルの
`random_seed` で固定しています。

### 検証項目

`validate_generated_data.py` は以下を自動チェックします。

- スペクトルが 50 個・各 201 点ある
- 結合エネルギーが単調減少・欠損値/無限値なし・強度が 0 以上
- 真のピーク面積比の合計が 1
- 処理が強いほど酸素含有ピーク比が平均的に増加（C–C/C–H は減少）
- 同一条件内にも十分なばらつきがある
- 同じシードで同じデータを再生成できる（再現性）

## 7. 完成条件（達成済み）

1. 50 個の独立した C1s スペクトルが生成されている
2. 通常試料は 3 ピークモデルで適切に再現できる
3. 処理条件に応じて C–O・O–C=O の面積比が平均的に増加する
4. 条件内の試料間ばらつきが存在する
5. 5 個の異常スペクトルが含まれる
6. 生成時の全パラメータが真値として保存される
7. 乱数シードによって完全に再現できる
8. 元データ・メタデータ・真値が明確に分離されている
9. 後続のフィッティングコードが真値を参照せず解析できる
10. フィッティング結果と真値を後から定量比較できる
